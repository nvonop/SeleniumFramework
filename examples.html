Framework Examples

These examples have been taken from the framework tests to show how easy it is to initialise the framework and start automating tests

Set System Properties

This example shows how to set the system properties in Groovy using Spock.
def setupSpec() {
System.setProperty("BROWSER", "Internet Explorer")
System.setProperty("VERSION", "8")
System.setProperty("PLATFORM", "WINDOWS")
System.setProperty("LOCAL_DRIVER", "true")
System.setProperty("REMOTE_DRIVER", "false")
System.setProperty("SAUCE_DRIVER", "false")
System.setProperty("webdriver.ie.driver", "C:\\Automation\\lib\\IEDriverServer.exe")
System.setProperty("IGNORE_SECURITY_DOMAINS", "true")
}

Initialise a browser and open a page

This example shows how to initialise a web browser in Groovy using Spock. The browser is initialised using the system properties above.
Browser browser;
def setup() {
browser = new Browser();
browser.open();
controlsPage = (ControlsPage)browser.navigate(ControlsPage.class);
}
 The browser navigates to the URL contained within the ControlsPage class and returns a page object corresponding to that page.

Close a browser

This example shows how to close a web browser in Groovy using Spock.
def cleanup() {
browser.close()
}

Take a screenshot

This example shows how to take a screenshot
browser.getScreenshot()
 Simply include this line at any point where you want to take a screenshot of the current browser window.

Click a button

This example shows how to click a button that has been modelled within a page object
Button button = new Button(driver, By.cssSelector("#buttonId"));
button.click()
