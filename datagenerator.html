The Data Generator

The purpose of this component is to provide a consistent way of generating data for a test. The classes within this package provide common methods for obtaining random strings, dates, day, month and year values, as well as email addresses, both valid and invalid according to different criteria.

This ensures that tests that use these methods will always have reliable, unique test data for every test run.

This component can be extended/enhanced in a production setting to create business specific test data e.g. a customer or other complex data type should the need arise.
