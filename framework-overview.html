Framework Overview

There are 5 main components to the framework:

Core

This consists of the Browser and ObjectMap classes. These provide access to high level web browser functions and a central place to store locators respectively.

Action Bot

Controls

This consists of the ActionBot class. It provides a quick way of performing command based actions on UI controls without having to model each individual control as a WebElement.
